# extensions-monitor README

This is an extension intended for use in environments where the user of the IDE should be prevented from using some particular extensions. Such situations include academic exams of technical interview evaluations.

The extension requires to buy a subscription, DO NOT INSTALL IT otherwise, it may block your IDE.

## Features

This extension monitors unwanted extensions in the context of an exam for example, to prevent students from using AI copilots.


## Requirements

You need to buy a yearly subscription from Patreon to get valid credentials (do not install it if you do not have valid credentials, otherwise, it may block your IDE because of unwanted extensions already installed):

## Extension Settings

You can add black-listed or white-listed extensions through the settings, but some AI copilot extensions cannot be activated from VSCode settings.
